# Prune

Prune a directory of files based on ctime keeping a specified number of files per day, week, month, and year.


## Usage

```
usage: prune.py [-h] [--days DAYS] [--weeks WEEKS] [--months MONTHS] [--years YEARS] [--dry-run] directory

positional arguments:
  directory

options:
  -h, --help       show this help message and exit
  --days DAYS
  --weeks WEEKS
  --months MONTHS
  --years YEARS
  --dry-run
```
